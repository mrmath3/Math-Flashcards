# Math Flashcards

A long-term system for building, organizing, and publishing math flashcards across multiple study platforms.

**One master spreadsheet → multiple finished products**

Platforms supported: Anki · Blooket · Gimkit · Quizizz · Quizlet

---

## About This Project

This project functions as a **math flashcard engine** — not a collection of one-off decks. All content originates from a single master spreadsheet and is exported to each platform from filtered subsets. GitHub serves as the central hub for source files, exported files, and published links.

For source files, export scripts, and project documentation, see the [repository](https://github.com/mrmath3/Math-Flashcards).
